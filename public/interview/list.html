计算规则如下:
1、面试数 = 通过数 + 淘汰数 + offer数  
2、总数 = 通过数 + 淘汰数 + ffer数 + 转移数（转发给其他人进行面试）+ 未面数（已分配，但没有面试）
3、淘汰率 = (淘汰数 / 面试数)  
4、通过率 = (通过数 / 面试数)
5、Offer率 = Offer数 / 面试数  
6、转移率 = 转移数 / (总数 - 未面数)
